# WebDrop 🛡️ (Final Release)

**安全・高速・プレミアムなP2Pファイル転送アプリケーション**

WebDropは、サーバーにデータを保存せず、端末同士が直接通信してファイルを送受するツールです。
最新の "Midnight" テーマを採用し、プロフェッショナルな見た目と操作性を実現しました。

---

## ✨ 主な機能 (Features)

### 1. 超高速ファイル転送
*   **P2P通信 (WebRTC):** サーバーを経由せず、端末間で直接データを送ります。
*   **大容量対応 (Streaming):** GBクラスの動画ファイルもメモリを圧迫せずに安定送信。
*   **複数ファイル:** ドラッグ＆ドロップでフォルダごとの送信も可能です。

### 2. 圧倒的な利便性
*   **QRコード接続:** スマホでスキャンするだけで、IP入力不要で即座につながります。
*   **クリップボード送信 (Paste-to-Send):** PCでスクリーンショットを撮り、`Ctrl+V` を押すだけで送信できます。
*   **PWA対応:** 「アプリとしてインストール」が可能。スマホのホーム画面から起動できます。
*   **自動再接続:** Wi-Fiが切れても、自動で復帰を試みます。

### 3. セキュリティ
*   **エンドツーエンド暗号化:** 通信経路は全てDTLS/SRTPで暗号化されます。
*   **ルーム分離機能:** Room IDを知っているユーザー同士だけがつながります。

## 🚀 使い方 (Quick Start)

### 1. サーバー起動
```bash
uvicorn server:app --host 0.0.0.0 --port 8001
```

### 2. PCから接続
ブラウザで [http://localhost:8001](http://localhost:8001) を開きます。
「Room ID」を入力して `Join Room` を押します。

### 3. スマホから接続 (QRコード)
PC画面のQRコードを読み取ると、自動的に同じ部屋に接続されます。

### 4. ファイルを送る
*   **ドラッグ＆ドロップ:** ファイルを画面に落とすだけ。
*   **コピペ送信:** 画像をコピーして `Ctrl+V`。
